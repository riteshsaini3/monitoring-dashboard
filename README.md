# Flask System Monitoring Dashboard

A real-time **System Monitoring Dashboard** built using **Python Flask + psutil + Chart.js** to track CPU and Memory usage visually in the browser.

---

##  Features

 Live CPU usage monitoring  
 Live Memory usage monitoring  
 REST API for metrics  
 Real-time charts using Chart.js  
 Lightweight Flask backend  
 Deployable on AWS EC2  
Beginner-friendly project structure  

---

##  Tech Stack

- Python
- Flask
- psutil
- HTML
- JavaScript
- Chart.js
- Git & GitHub
- AWS EC2 (Deployment)
